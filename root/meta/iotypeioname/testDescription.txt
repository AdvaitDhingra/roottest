This test aims to read a rootfile written with Root5 and using a selection xml
which featured an iotype tag.
The classes.h file was identical.
The selection xml file was the following: 

<lcgdict>
   <class name="Container" />
   <class name="SuperCluster">
      <field name="energy" iotype="Double32_t" />      
   </class>
   <class name="Particle">
      <field name="vertex" iotype="ROOT::Math::PositionVector3D<ROOT::Math::Cartesian3D<Double32_t>,ROOT::Math::DefaultCoordinateSystemTag>" />
   </class>
</lcgdict>

The macro to dump on the file was the following:

void writeFile(){
   
   {
   gSystem->Load("libCintex.so");
   ROOT::Cintex::Cintex::Enable();
   }
   Container cont;
   TFile ofile("example.root","RECREATE");
   cont.Write();
   ofile.Close();
   
}

The command to build the dictionary and dump on the file was:
genreflex classes.h  -s classes_selection.xml --rootmap al5.rootmap -I $ROOTSYS/include 
clang++ -fPIC -shared -o libclasses.so classes_rflx.cpp `root-config --cflags --libs` -lReflex -lCintex
root -b -q writeFile.C 

The version of ROOT was 
ROOT 5.34/17 (v5-34-17@v5-34-17, Feb 25 2014, 15:41:27 on linuxx8664gcc)
